🍛 HomeMeals - Home Cook Marketplace

A full-stack web application that connects home cooks with hungry neighbors.
Home cooks (aunties, paatis, ammas) post their daily menu and people in the
area can browse and order fresh homemade food.

🚀 Features
 Browse daily menus from local home cooks
 Filter by Location, Community (Tamil / Muslim / Hindu) and Food Type (Veg / Non-Veg)
 Add to cart and place orders with delivery address
 Home cooks can register and post their daily menu
 Real-time quantity updates when orders are placed
 Fully responsive full-screen UI

🛠️ Tech Stack
Frontend:React.js, Vite
Backend:Node.js, Express.js
API:RESTful API with GET & POST endpoints
Styling:CSS-in-JS (inline styles)

 📦 Installation

Backend
cd backend
npm install
node server.js

 Frontend
cd frontend
npm install
npm run dev

🌐 Live
 Frontend: http://localhost:5173
 Backend API: http://localhost:5000

💡 Idea
Inspired by the real problem of home cooks (especially women) who have
amazing cooking skills but no platform to reach customers. HomeMeals
gives them a digital storefront with zero setup cost.
